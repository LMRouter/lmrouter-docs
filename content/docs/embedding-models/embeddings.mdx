---
title: Create Embeddings
description: Generate embeddings with LMRouter
---

The embeddings endpoint provided by LMRouter is compatible with the OpenAI embeddings endpoint. It is available at

```
https://api.lmrouter.com/openai/v1/embeddings
```

Please refer to the [OpenAI embeddings endpoint](https://platform.openai.com/docs/api-reference/embeddings) for more information on how to use the embeddings endpoint.
