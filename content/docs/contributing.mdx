## Contributing to LMRouter

Thank you for your interest in contributing to LMRouter! The source code for LMRouter is available on [GitHub](https://github.com/LMRouter/lmrouter). You are welcome to contribute to LMRouter by submitting issues, feature requests, or pull requests.

## Add Support for New Models and Providers

To add support for a new model or provider, you can submit a pull request to the [LMRouter models repository](https://github.com/LMRouter/lmrouter-models).

* Existing models config can be found here: https://github.com/LMRouter/lmrouter-models/tree/main/library/models.
* Existing providers config can be found here: https://github.com/LMRouter/lmrouter-models/tree/main/library/providers.

## Contributing to This Doc

The source code for this documentation is available on [GitHub](https://github.com/LMRouter/lmrouter-docs). You are welcome to contribute to this documentation by submitting issues, feature requests, or pull requests.
