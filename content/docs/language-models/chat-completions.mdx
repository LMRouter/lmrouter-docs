title: Create Chat Completions
description: Generate chat completions with LMRouter
icon: MessageCircle
---

The chat completions endpoint provided by LMRouter is compatible with the OpenAI chat completions endpoint. It is available at

```
https://api.lmrouter.com/openai/v1/chat/completions
```

Please refer to the [OpenAI chat completions endpoint](https://platform.openai.com/docs/api-reference/chat/create) for more information on how to use the chat completions endpoint.
