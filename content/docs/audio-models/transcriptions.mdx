---

The transcriptions endpoint provided by LMRouter is compatible with the OpenAI transcriptions endpoint. It is available at

```
https://api.lmrouter.com/openai/v1/audio/transcriptions
```

Please refer to the [OpenAI transcriptions endpoint](https://platform.openai.com/docs/api-reference/audio/createTranscription) for more information on how to use the transcriptions endpoint.
