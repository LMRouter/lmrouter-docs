The speech endpoint provided by LMRouter is compatible with the OpenAI speech endpoint. It is available at

```
https://api.lmrouter.com/openai/v1/audio/speech
```

Please refer to the [OpenAI speech endpoint](https://platform.openai.com/docs/api-reference/audio/createSpeech) for more information on how to use the speech endpoint.
