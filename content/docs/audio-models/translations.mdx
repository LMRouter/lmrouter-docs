The translations endpoint provided by LMRouter is compatible with the OpenAI translations endpoint. It is available at

```
https://api.lmrouter.com/openai/v1/audio/translations
```

Please refer to the [OpenAI translations endpoint](https://platform.openai.com/docs/api-reference/audio/createTranslation) for more information on how to use the translations endpoint.
